# 学生使用 Claude Code 提交作业

本教程用于在本机 Claude Code 终端中提交 Vibe Coding 作业。

## 开始前准备

你需要从课程方获得：

- 课程客户端安装包或安装命令；
- 课程邀请码；
- 作业编号；
- Claude Code 本地终端环境。

不需要安装 Docker，也不需要填写服务器地址。

## 第一次安装与注册

1. 按课程方提供的安装说明安装客户端。安装完成后，在新开终端执行：

   ```powershell
   vibe-submit --help
   ```

   能看到命令帮助即表示安装成功。

2. 在终端执行一次注册：

   ```powershell
   vibe-submit setup
   ```

3. 根据终端提示输入课程邀请码、学号、姓名和密码。

密码只在你的本机终端输入，不会发送到 Claude 对话中。注册成功后，同一台电脑通常不需要重复注册。

## 提交作业

1. 打开终端，进入**包含作业代码的文件夹**。例如：

   ```powershell
   cd <你的作业文件夹>
   claude
   ```

   请确认该文件夹确实包含你的代码文件。提交工具会收集这个文件夹及其子文件夹中符合规则的项目文件。

2. 在 Claude Code 当前会话中输入：

   ```text
   /submit-homework <作业编号>
   ```

3. 阅读预览信息。系统会提交：

   - 当前作业目录中符合规则的项目文件；
   - 当前 Claude Code 会话内容；
   - 项目中的截图和最终报告文件（如有）。

   系统不会收集其他项目、其他 Claude 会话、Git 历史、依赖目录或常见敏感文件。

4. 只有在确认内容无误后，明确回复确认提交。未确认时不会上传。

5. 看到“提交成功”后，保留终端输出作为记录。

## 重新提交

如果作业已经提交过，系统会保护原提交，避免误覆盖。

1. 确认你已经在正确作业目录中重新启动 Claude Code。
2. 再次执行 `/submit-homework <作业编号>`。
3. 只有你明确要求覆盖时，系统才会上传新版本。

教师端默认查看最新版本；旧版本保留为提交历史。

## 常见问题

### 找不到 `/submit-homework`

关闭并重新启动 Claude Code。如果仍找不到，请重新执行课程方提供的客户端安装步骤。

### 提交后教师端没有看到代码

通常是因为 Claude Code 启动位置不对。退出 Claude Code，先在包含作业代码的目录执行 `cd <作业文件夹>`，再运行 `claude` 后重新提交。

### 网络失败

上传包会保存在本机等待重试。网络恢复后，在终端运行：

```powershell
vibe-submit retry
```

### 会话校验失败

请从作业目录重新启动 Claude Code，并在该会话中提交；不要从其他目录启动后再切换项目。
